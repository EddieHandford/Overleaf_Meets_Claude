\begin{abstract}
\noindent This guide describes a three-tool workflow for writing \LaTeX{} documents: drafting in Overleaf for its live preview and collaboration, versioning through GitHub, and using Claude Code for bulk, structural, and cross-cutting edits that are slow or error-prone to do by hand. The repository containing this guide is itself an example of the workflow it describes.
\end{abstract}
